## 🎮 How to Use

### Quick Start: Bulk Rename (5 minutes)
```
1. Click "Browse" → Select folder
2. Click "Load All" → Files load instantly ⚡
3. Click "Select All" → All files checked & highlighted ✓
4. Enter rename pattern: {name}_new{n}
5. Click "Rename" → Done!
```

### Traditional: Find Duplicates (3 minutes)
```
1. Click "Browse" → Select folder
2. Click "Scan" → Find duplicates
3. Click "Cleanup All" → Remove copies
4. Done!
```
